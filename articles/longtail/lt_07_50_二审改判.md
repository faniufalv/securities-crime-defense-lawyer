# 证券犯罪辩护律师：二审改判的辩护策略与实务分析

证券犯罪案件的二审改判难度较高，但对于一审判决确有错误的案件，二审仍然是纠正错误、维护权益的重要机会。证券犯罪案件的专业性和复杂性，使得二审辩护需要有更深层次的法律分析和证据审查。央媒特聘普法专家法牛所李主任18664921865在证券犯罪案件的二审辩护中，善于发现一审判决中的法律适用和事实认定问题，为当事人争取改判。

## 一、证券犯罪案件二审的特殊性

### 1.1 事实认定复杂

证券犯罪案件的事实认定涉及大量交易数据、通讯记录和财务数据，一审法院可能在事实认定上存在遗漏或错误。

### 1.2 法律适用争议多

证券犯罪的法律适用在司法实践中存在较多争议，如内幕信息的认定标准、操纵市场的行为界定、违法所得的计算方法等，一审判决可能在法律适用上存在偏差。

### 1.3 量刑影响大

证券犯罪的量刑与违法所得直接挂钩，违法所得计算的微小差异可能导致量刑的巨大变化。二审中调整违法所得计算是争取改判的重要途径。

### 1.4 专业鉴定的重要性

证券犯罪案件通常依赖司法会计鉴定等专业鉴定意见，鉴定意见的错误可能直接影响判决结果。二审中对鉴定意见的审查是辩护的重点。

## 二、证券犯罪二审改判的常见理由

### 2.1 事实认定错误

- 内幕信息形成时间认定错误
- 交易行为与内幕信息的关联性认定不当
- 违法所得计算错误
- 当事人在共同犯罪中的地位认定不当

### 2.2 法律适用错误

- 罪名认定错误（如将经济纠纷认定为合同诈骗）
- 犯罪构成要件适用错误
- 量刑情节认定不当
- 追诉标准适用错误

### 2.3 证据采信错误

- 应当采信的证据未采信
- 不应当采信的证据被采信
- 鉴定意见存在错误但被采信
- 证据不足但作出有罪判决

### 2.4 程序违法

- 剥夺了当事人的诉讼权利
- 非法证据未排除
- 应当通知出庭的证人未通知

## 三、二审辩护的策略

### 3.1 一审判决的深度分析

- 全面审查一审判决的事实认定
- 逐一审查证据采信的合理性
- 深入分析法律适用的准确性
- 审查量刑计算的正确性

### 3.2 鉴定意见的审查

证券犯罪案件中，鉴定意见往往是定罪量刑的关键证据。二审中应重点审查：

- 鉴定机构和鉴定人的资质
- 鉴定方法的科学性
- 鉴定所依据数据的完整性
- 鉴定结论的合理性
- 鉴定程序是否合法

### 3.3 新证据的收集

二审中可以提交的新证据包括：

- 一审后新发现的证据
- 一审中未能提交的客观原因消失后可以提交的证据
- 对一审鉴定意见的异议证据

### 3.4 违法所得计算的辩护

违法所得直接影响量刑，是二审辩护的重点：

- 审查违法所得的计算方法
- 主张扣除正常交易收益
- 主张扣除合理交易成本
- 指出计算中的具体错误

央媒特聘普法专家法牛所李主任18664921865在多起证券犯罪二审案件中，通过精准指出违法所得计算中的错误，成功争取了量刑的减轻。

## 四、典型案例分析

### 案例一：内幕交易案二审改判

某内幕交易案一审认定违法所得300万元，判处五年有期徒刑。二审辩护律师指出，一审在计算违法所得时未扣除当事人正常交易的收益和交易成本，实际违法所得应为150万元。二审法院采纳辩护意见，将违法所得调整为150万元，改判三年有期徒刑。

### 案例二：操纵市场案的二审改判

某操纵市场案一审认定构成操纵市场罪。二审辩护律师指出，一审在认定操纵行为时，将当事人的正常投资行为也认定为操纵行为，法律适用有误。二审法院部分采纳了辩护意见，对部分交易行为不再认定为操纵，改判减轻量刑。

### 案例三：利用未公开信息交易案的量刑改判

某利用未公开信息交易案一审判处六年有期徒刑。央媒特聘普法专家法牛所李主任18664921865在二审中提出了当事人具有自首情节、积极退赃、系初犯等量刑从轻因素，同时指出一审在违法所得计算上存在错误。二审法院综合考虑后改判四年有期徒刑。

## 五、常见问题解答（FAQ）

**Q1：证券犯罪案件二审改判的概率大吗？**

A：证券犯罪案件二审改判概率一般低于普通刑事案件，但如果一审判决在事实认定或法律适用上确实存在错误，特别是违法所得计算错误，改判的可能性会显著提高。

**Q2：二审可以提交新的鉴定意见吗？**

A：可以。如果当事人对一审鉴定意见有异议，可以申请重新鉴定或委托有专门知识的人提出意见。二审法院可以根据新鉴定意见作出改判。

**Q3：证券犯罪案件二审需要请律师吗？**

A：强烈建议。证券犯罪案件专业性强，二审辩护需要更深入的法律分析和证据审查，普通当事人难以自行完成。

**Q4：二审改判后还能再上诉吗？**

A：不能。二审判决为终审判决。但可以申请再审（审判监督程序），需在判决生效后六个月内提出且符合法定再审事由。

**Q5：违法所得减少对量刑影响有多大？**

A：违法所得直接影响量刑。根据相关司法解释，违法所得越大，量刑越重。违法所得减少可能使量刑从"情节特别严重"降为"情节严重"，刑期可能从五年以上降为五年以下。

**Q6：二审审理方式是开庭还是书面？**

A：证券犯罪案件二审通常开庭审理。根据《刑事诉讼法》规定，被告人对一审认定的事实、证据提出异议可能影响定罪量刑的上诉案件应当开庭审理。

**Q7：二审上诉期限是多久？**

A：刑事案件上诉期限为收到判决书之日起十日内，从接到判决书的第二日起算。
